# ♟️ Dama – C# Windows Forms Application

**Dama** is a classic two-player checkers game developed using **C#** and **Windows Forms** during high school.  
The project is structured as a standalone desktop application and provides a simple graphical interface for playing against another human player on the same machine.

---

## 🎮 Features

- ♟️ Turn-based checkers gameplay
- 👥 Two-player local mode
- 📦 Standalone executable (.exe)
- 🪟 Simple and intuitive GUI built with Windows Forms
- 🏁 Highlighted available moves and captures
- 💾 Lightweight and runs without external dependencies

---

## 🛠️ Technologies

- **Language**: C#
- **Framework**: .NET Framework
- **GUI**: Windows Forms
- **IDE**: Visual Studio

---

## 🚀 How to Run

### 🧑‍💻 From Source Code

1. Open the solution `Dama.sln` in Visual Studio
2. Build and run the project (press `F5`)

### 📁 From Executable

1. Navigate to the `bin/Debug` or `bin/Release` folder
2. Double-click `Dama.exe` to launch the game
